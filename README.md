# ⚡ Calculadora de Consumo de Energia

## 📌 Objetivo
Este projeto tem como objetivo calcular o consumo mensal de energia elétrica de um aparelho, com base na sua potência e tempo de uso diário.

## 💻 Linguagem utilizada
- Python 🐍

## 🧮 Fórmula utilizada
consumoMensal = (potencia * horasDia * 30) / 1000

## ▶️ Como executar o programa

1. Ter o Python instalado
2. Baixar este repositório
3. Abrir o terminal na pasta do projeto
4. Executar o comando:

python app.py

## ⚡ Funcionalidades
- Calcula o consumo mensal em kWh
- Mostra o resultado formatado
- Calcula o custo estimado com base no valor do kWh
